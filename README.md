# Safora Contact Form Automation Test

Uploaded codes contains a simple automation test script for the Safora web page Contact Us form using Selenium and Python.

## Features

- Opens the Safora website
- Navigates to the Contact Us page
- Tests form validation
- Verifies required field behavior
- Uses Selenium WebDriver with Firefox

## Technologies Used

- Python
- Selenium
- Firefox for browser

## Installation

Install Selenium:

```bash
pip install selenium
```
## Run the Test

```bash
python submit_with_valid_inputs.py
python submit_without_inputs.py
```

## Test Scenario

1. Open Safora website
2. Navigate to Contact Us page
3. Attempt form submission without inputs
4. Verify validation messages are displayed
